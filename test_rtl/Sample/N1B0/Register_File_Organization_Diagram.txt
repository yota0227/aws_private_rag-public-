                          KEY DESIGN FEATURES

✓ DUAL BANKING: All three RFs (DEST, SRCA, SRCB) use 2 banks for double-buffering
✓ LATCH ARRAYS: All use latch technology (not SRAM) for zero-latency read-modify-write
✓ WORD ORGANIZATION:
  - DEST: 4 columns (matches FP-Tile output width)
  - SRCA: 16 words/row (supports K-tiling for K=8192 GEMM)
  - SRCB: 16 datums/row (matches FPU output granularity)
✓ ADDRESSING: 6-bit address for both SRCA (log₂48) and SRCB (log₂64)
✓ DATUM WIDTH: All use 16-bit datum width (half-float or INT16 or INT8×2)
✓ ZERO-LATENCY: Combinational read-modify-write in same cycle (critical for accumulation)
