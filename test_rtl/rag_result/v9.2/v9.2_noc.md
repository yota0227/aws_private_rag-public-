## 1. Overview

Trinity N1B0 employs a 2D mesh Network-on-Chip (NoC) fabric connecting all 20 tiles in a 4×5 grid. The NoC supports three routing algorithms and provides reliable data transport with ECC protection.

Key characteristics:
- 4×5 mesh topology (NumAxes=2, NumDirections=2)
- 3 routing algorithms: DIM_ORDER, TENDRIL, DYNAMIC
- EnableDynamicRouting = 1'b1
- SECDED ECC (116-bit data, 10-bit check)
- Inter-column repeaters for signal integrity
- Asynchronous clock domain crossing support

---

## 2. Routing Algorithms

| Algorithm | Type | Description |
|-----------|------|-------------|
| DIM_ORDER | Deterministic | XY dimension-ordered routing (route X first, then Y) |
| TENDRIL | Adaptive | Congestion-aware adaptive routing |
| DYNAMIC | Dynamic | Runtime-selectable path (EnableDynamicRouting=1) |

## 9. Inter-column Repeaters (v9.2 new)

- tt_noc_repeaters_cardinal placed between columns for signal integrity
- Connects input/output NoC packages across column boundaries
- Located at Y=3, Y=4 positions (between NOC2AXI and ROUTER tiles)
