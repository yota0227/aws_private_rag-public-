## 1. Overview

EDC1 (Error Detection and Correction, version 1) is the on-chip error management subsystem for Trinity N1B0. It implements a serial ring bus connecting all tiles for error reporting, detection, and correction signaling.

Key characteristics:
- Toggle-handshake serial protocol
- 16-bit data + parity
- U-shape ring topology
- APB4 slave interface for configuration
- 5 IRQ output lines
- Harvest bypass for defective tile isolation
