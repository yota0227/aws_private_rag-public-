## 1. Overview

*Source: HDD [1]*

The EDC subsystem includes a block-level component serving as part of the EDC functionality. EDC1 modules use `tt_edc1_*` naming convention.

**KB-confirmed:** BIU uses APB4 protocol with 5 interrupt outputs; NoC security block registers; serial bus repeaters with assertion-based verification.
