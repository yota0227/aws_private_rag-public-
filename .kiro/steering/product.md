# Product Overview

BOS-AI Private RAG is an air-gapped, enterprise Retrieval-Augmented Generation system for a semiconductor design company (BOS Semi). It enables engineers to query internal technical documents (RTL code, design specs, architecture docs) using natural language via an on-premises chatbot (Obot + MCP protocol), with answers generated by AWS Bedrock (Claude) backed by OpenSearch Serverless vector search.

## Key Characteristics

- Fully private: no internet gateway, no NAT, no public endpoints. All traffic stays within VPN + VPC Endpoints + VPC Peering.
- Hybrid architecture: on-premises network (192.128.0.0/16) connects to AWS via IPsec VPN through Transit Gateway.
- Multi-region: Seoul (ap-northeast-2) handles frontend/API/VPN connectivity; Virginia (us-east-1) hosts Bedrock KB, OpenSearch Serverless, and document storage.
- Document pipeline: upload via web UI → S3 (Seoul) → cross-region replication → S3 (Virginia) → Bedrock KB ingestion → OpenSearch vector indexing.
- Domain: rag.corp.bos-semi.com (Route53 Private Hosted Zone, DNS conditional forwarding from on-prem).
- Project tag: BOS-AI
- Environment: Production

## Primary Users

- Semiconductor design engineers querying specs and RTL documentation
- IT/DevOps team managing infrastructure and deployments
